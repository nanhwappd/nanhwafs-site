# 南华新官网总规划 v1.0（2026-07-06 · Fable 5 全局规划）

## 0. 一句话定位

把 **8 年脸书全档（2018–2026，1241 篇帖子 + 8484 媒体文件）** 变成官网的
「按年新闻馆 + 数字校史馆」内容底盘，赶在 **8 月 29 日 90 周年校庆** 前上线主体，
之后官网成为新闻首发地、脸书降级为转发渠道（原生内容利 SEO/GEO）。

## 2. 架构决策（ponytail：能不建的不建）

- **纯静态 + GitHub Pages 不变**。1241 页对静态站是小数目，无需 CMS/数据库。
- **媒体走「精选入站」**：新闻页每篇配 3–8 张代表图（1600px 长边压缩后 ~150–250KB/张），
  预计 ~3000 张 ≈ 600–800MB，单 repo 装得下。**6GB 全档不进 repo**，留 D:\ + Google Drive 镜像当母本。
  - 若日后超 1GB：按年代拆媒体仓（`nanhwappd/fb-media-20xx`）跨仓引用，方案已备不先建。
- **视频不进 repo**（GH 100MB/文件硬限，云果展已验证路线）：166 个 mp4 精选上传 YouTube 频道后内嵌，其余留档。
- **繁简策略（已裁 2026-07-06）**：全站**简体**。历史帖文用 OpenCC 脚本批量转换（零 token），马来西亚简繁通用，不构成问题。

## 3. 分期路线图

### P1 解析器 + 内容普查（本周，纯本地，零风险）
- 一支 Python 脚本：`profile_posts_1.html` → `posts.json`
  （每篇：日期 / 全文 / 媒体路径列表 / 类型 / 相册归属）。
- 产出《内容普查报告》：按年 × 按类（新闻报道 / 通告 / 招生 / 成绩喜报 / 纯转发）计数，
  标出「值得成页」清单——这是后面一切工序的量尺。
- **一鱼两吃**：同一份 `posts.json` 直接喂 nanhwa-wiki 的脸书语料逐年导入计划
  （corpus 配套遗产 + 新闻写作指纹 v2），不必二次解析。

### P2 信息架构定稿（与 P1 并行）
- `sitemap.md`：独中官网八栏基线（关于/新闻/招生/学术/联课/校友/联系/捐助）
  + 两个特色栏位：**按年新闻馆**（2018–2026）+ **数字校史馆**（90 周年主打，时光隧道相册是现成素材）。
- 锁新闻页模板（一次定稿，1200+ 页共用）；修「诗巫→曼绒」。

### P3 媒体管道（P1 报告出来后）
- Pillow 批量压缩精选图；按 `news/YYYY/media/` 落位。
- 视频清单交由用户/冠铭决定哪些上 YouTube。

### P4 按年新闻馆批量生成（体力活，可派施工窗口）
- 生成器脚本：`posts.json` + 模板 → `news/2018/…2026/` 每篇一页 + 年度索引页。
- 顺序建议**倒序**（2026→2018）：新内容 SEO 价值最高，先上线先收录。
- 每完成一年即 push 上线，不憋大招。

### P5 SEO / GEO / 域名（P4 过半即可启动）
- `sitemap.xml` + 每页 meta/OG + 结构化数据（Article/School）。
- 旧站清理：site123 草案撤下、gbs2u 加 noindex（ACTIVE.md 既有待办）。
- **域名（待裁）**：绑 `nanhwafs.edu.my` 子域（如 `www.`）需校方 DNS 权限；
  谈妥前 GH Pages 原址先跑，绑定后 301 无损迁移。

### P6 增量运营闭环（上线后）
- 新闻新流程：/news 起稿 → 官网原生页首发 → FB 贴摘要 + 链接回官网。
- 每季度一次 FB 导出增量补档（同一支 P1 解析器可复用）。

## 4. 裁决记录（2026-07-06 林子策）

1. **繁简**：✅ 全站简体，OpenCC 脚本转换。
2. **迁移量级**：全量脚本路线比逐篇精选省 10–20 倍 tokens（生成是脚本活，精选才要判断）。
   方案：全量 + 机械过滤（「shared a link/post 且无正文」的 ~63 篇水贴剔除，其余全收；
   配图规则：帖子自带图 ≤8 全收、超了取前 8）。事后如要打标精选，走 Gemini 管道不烧 Claude。
3. **域名**：✅ 不急。DNS 不影响架设，唯一节点=P5 提交 sitemap 前须绑好（否则 github.io 地址
   先被收录、绑域后重爬白等几周）。「向校方拿 DNS」挂为 **P5 前置闸门**，P4 过半（约 8 月初）再要。
4. **视频**：✅ 学校有 YouTube 频道（冠铭账号）。P3 视频精选清单交冠铭上传。
